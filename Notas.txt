Ejecucion del Webpack:

Antes de ejecutar comandos asegurarse que la terminal se encuentre en el directorio actual del proyecto: 
C:\Users\carlo\Desktop\Javascript\07. App - To-Do

1. Ejecutar el comando "npm install" para instalar los modulos de Node
2. Ejecutar el comando "npm run build" para que cree la carpeta dist
   con los archivos index.html, js y css optimizados
   para pasar a produccion.
3. Ejecutar el comando "npm start" para que abra la aplicacion
   en el navegador

Nota: Para reutilizar el webpack no borrar la imagen del asset, da error. 
Es decir, debe ir una imagen x obligatorio

Despliege a GitHub

1. git init
2. git add .					   //Toma una foto de todos los archivos del proyecto que sirve para recuperarlas en un futuro					
3. git commit -m "Primer Commit"                   //Guarda en Git todos los cambios del proyecto


Generando el archivo dist, que es el archivo de javascript, hmtl, css optimizado que se va a desplegar en produccion:

4. npm run build
5. En caso de haber un archivo docs borrarlo
6. renombrar el archivo 'dist' por 'docs' ---> Archivo que se utilizara para desplegarlo a github

Volver a guardar los cambios en git

7. git add .
8. git commit -m "Primer Commit"
